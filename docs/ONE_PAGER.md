# Byte-sized overview for hiring managers

**ProcureIQ** is a working procurement analytics platform that shows how I design data infrastructure and ship analytics products—not just models.

**In one demo you’ll see:**
- Multi-system spend integration → governed lakehouse
- Spend cube with Direct/Indirect hierarchies
- Rate leakage, should-cost gap, TCO, finance budget variance
- Supplier risk + anomaly detection
- Semantic self-serve Q&A (AI-ready)

**Stack:** Python · DuckDB · SQL dimensional modeling · scikit-learn · Streamlit/Plotly · YAML semantic layer  

**Transferable to:** Microsoft Fabric, Power BI, Snowflake, Databricks, AWS.
